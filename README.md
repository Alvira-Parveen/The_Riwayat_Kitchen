# 🥘 The Riwayat Kitchen

Welcome to the official repository for **The Riwayat Kitchen**, a premium cloud kitchen based in Muzaffarpur, Bihar. This project serves as a digital landing page and interactive menu, showcasing our authentic, home-cooked delicacies.

---

## 🔗 Live Site
Check out the live website here: [https://alvira-parveen.github.io/The_Riwayat_Kitchen/](https://alvira-parveen.github.io/The_Riwayat_Kitchen/)

---

## 📝 About the Project
"Riwayat" — the stories, traditions, and flavors that live on through generations. This project was built to bring our home-style recipes to the digital world. 

The website is designed to be:
*   **Responsive:** Looks great on mobile, tablet, and desktop.
*   **User-Friendly:** Simple navigation to explore our menu.
*   **Action-Oriented:** Features direct links to order via WhatsApp, Zomato, and Swiggy.

---

## 🛠️ Tech Stack
*   **HTML5** - For structure.
*   **CSS3** - For styling and responsive design.
*   **JavaScript** - For interactive elements.

---

## ✨ Key Features
*   **Full Menu Display:** Categorized sections for Chicken Specials, Burgers (Veg & Non-Veg), and Indian Delights.
*   **Order Integration:** Seamless links to external delivery platforms (Zomato/Swiggy) and direct WhatsApp ordering.
*   **About Us:** A story-driven section highlighting the heritage and quality of our food.
*   **Fast Loading:** Optimized for quick access even on slow internet connections.
